# Various PowerShell Scripts

A collection of useful PowerShell scripts that don't require a dedicated repository.

| Script | Description |
|--------|-------------|
| **Black&nbsp;Theme** | Applies a true black theme to Windows. |
| **Core&nbsp;Parking** | Adjusts OS-managed CPU core parking. |
| **Digital&nbsp;Markets&nbsp;Act** | Enables EU DMA compliance, allowing native removal of Microsoft Edge and other enforced apps. |
| **Higher&nbsp;Scaling&nbsp;No&nbsp;Accel** | Removes mouse acceleration on the desktop when using scaling above 100%. |
| **Legacy&nbsp;File&nbsp;Explorer** | Restores the classic file explorer ribbon on Windows 11. |
| **Manage&nbsp;Windows&nbsp;Updates** | Pauses updates for 1 year or disables them entirely. |
| **Open&nbsp;File&nbsp;Security&nbsp;Warning** | Removes the security warning and automatic file blocking for downloaded files. |
| **Start&nbsp;Menu&nbsp;Version** | Activates the new 25H2 start menu or reverts to the 24H2 style on supported Windows 11 26200 builds. |
| **User&nbsp;Account&nbsp;Control** | Enables or disables UAC. |

> [!NOTE]
> All scripts are reversible; any change can be undone.
